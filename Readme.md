# 🚗 Car Price Prediction

## 📌 Overview

This project predicts the price of a car based on features like engine size, fuel type, and other specifications using Machine Learning.

## 🛠️ Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib

## ⚙️ Features

* Data cleaning and preprocessing
* Data visualization
* Model training using Linear Regression
* Prediction on test data

## 📊 Result

Achieved an accuracy of ** 0.5954654304679821%** on test data.

## ▶️ How to Run

1. Clone the repository
2. Install dependencies:

   ```
   pip install pandas numpy matplotlib scikit-learn
   ```
3. Run:

   ```
   python main.py
   ```

## 📁 Project Structure

* main.py → main code
* dataset.csv → dataset
* README.md → project documentation
